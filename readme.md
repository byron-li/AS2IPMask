# AS2IPMask:

从CAIDA获取最新pfx2as数据, 根据配置ASN, 输出IP地址段(掩码>=16)的xlsx格式文件.

## 配置

*	配置文件位于conf目录下
	*	config.ini: 程序配置文件, 指定获取对应ASN对应IP段: IPv4(4) /IPv6(6)/ 全部(all).
	*	as.conf: ASN配置文件, 输入RuleName及ASN, 逗号分隔. 一条规则下有多个ASN也需用逗号分隔.  
		e.g. test,4808,6836

## 输出

*	每次输出覆盖上一次生成结果, 文件名: AS_IP_mapping.xlsx
